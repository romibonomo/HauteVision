### **5. Taking Screenshots**

#### **Using Xcode Simulator:**
1. Open Xcode
2. Select target device (iPhone 14 Pro Max, iPad Pro, etc.)
3. Run the app (`Cmd + R`)
4. Navigate to desired screen
5. Take screenshot (`Cmd + S` or Device ‚Üí Screenshot)

#### **Using Physical Device:**
1. Take screenshots on device (`Volume Up + Power`)
2. Transfer to computer via AirDrop/iCloud

### **7. Best Practices**

#### **Do:**
- ‚úÖ Show real-looking medical data
- ‚úÖ Use consistent sample patient (John Smith)
- ‚úÖ Show different measurement types
- ‚úÖ Include charts and graphs
- ‚úÖ Show both eyes (OD/OS) when relevant
- ‚úÖ Display proper date ranges
- ‚úÖ Show medication information

#### **Don't:**
- ‚ùå Show empty states or "no data" messages
- ‚ùå Include placeholder "Lorem ipsum" text
- ‚ùå Show personal/private information

- ‚ùå Show loading screens

### **8. Disable Screenshot Mode**
1. Go to **Profile** tab
2. Turn off **"Screenshot Mode"** toggle
3. App returns to normal operation with real user data

### **9. Upload to App Store Connect**
1. Go to [App Store Connect](https://appstoreconnect.apple.com)
2. Select your app ‚Üí **App Store** tab
3. Upload screenshots for each required device size
4. Add captions describing each screenshot
